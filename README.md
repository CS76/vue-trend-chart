## Installation

```shell
npm i vue-trend-chart
```
## Usage

```js
import Vue from "vue";
import TrendChart from "vue-trend-chart";

Vue.use(TrendChart);
```

## Props

**`datasets`** **required**

*Type:* Array
*Description:* Array of objects with properties for each dataset

------------

**`grid`**

*Type:* Object
*Default:* null
*Description:* Grid properties
